SAÉ 12

S'initier aux réseaux informatiques

Objectif du cours

Découvrir les bases des réseaux : topologies, adressage IP, configuration de routeurs et commutateurs.

Projet réalisé

TP : Création d’un réseau local simple avec Packet Tracer :

  • Configuration de commutateurs Cisco
  • Affectation d’adresses IP fixes
  • Test de connectivité avec ping
  • Mise en place de VLAN simples

Compétences acquises

  • Comprendre le modèle OSI et les couches réseau
  • Savoir affecter une adresse IP statique à un hôte
  • Utiliser Packet Tracer pour simuler un réseau
  • Manipuler l’interface CLI d’un switch ou routeur Cisco
